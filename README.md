# Smart Ambulance Traffic Management System 🚑🚦

## Project Overview
This project is a Smart Traffic Management System developed using Python, Flask, Flask-SocketIO, and SQLite. It gives priority to ambulances by dynamically controlling traffic signals, reducing emergency response time.

## Features
- 🚑 Ambulance priority system
- 🚦 Real-time traffic signal management
- 👨‍⚕️ Authorized ambulance verification
- 📋 Driver log management
- 💾 SQLite database integration
- 🌐 Interactive web interface

## Technologies Used
- Python
- Flask
- Flask-SocketIO
- SQLite
- HTML
- CSS
- JavaScript

## Team Project
This project was developed as a group academic mini project.
